// how it works

Three steps. Seriously, that's it.

01
Get your API key
Enter your email below and receive a unique magno_sk_ key instantly. No credit card. No setup. Free forever up to 1,000 calls/month.
02
Store memories after each response
After your LLM responds, call POST /memory/store with the conversation text. MagnoAPI converts it to a vector and saves it permanently.
03
Retrieve context before each call
Before calling your LLM, call POST /memory/search with the user's message. Inject the results into your prompt. Your AI now remembers.
// integration

Two calls. Any language. Any LLM.

your_app.py 
# Before calling your LLM
memories = requests.post("magnoapi/memory/search",
  headers={"X-API-Key": "magno_sk_..."},
  json={
    "user_id": "user_123",
    "query":   user_message,
    "top_k":   3
  }
).json()["results"]

# Inject into your LLM prompt
context = "\n".join([m["text"] for m in memories])
prompt  = f"Context:\n{context}\n\nUser: {user_message}"

# After your LLM responds
requests.post("magnoapi/memory/store",
  headers={"X-API-Key": "magno_sk_..."},
  json={"user_id": "user_123", "text": conversation}
)
  • 🧠
    Semantic search, not keyword search
    MagnoAPI uses vector embeddings to find memories by meaning, not exact words. "I love Python" and "Python is my favourite language" both match.
  • ⚡
    Works with every LLM
    Claude, GPT-4, Gemini, Llama, Mistral — MagnoAPI sits between your app and any LLM. Completely model-agnostic.
  • 🔒
    Per-user memory isolation
    Every memory is scoped to a user_id. User A never sees User B's memories. Built-in privacy by design.
